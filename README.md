# Frost & Swirl Ice Cream Website

This project is a premium AI-generated ice cream website created as part of the Future Interns Prompt Engineering Internship. The website combines luxury dessert aesthetics, modern UI principles, and engaging visual storytelling to deliver a delightful digital experience for ice cream lovers.

## 🏆 Project Overview

This project showcases a visually stunning and modern website for **“Frost & Swirl”**, a premium ice cream lounge located in Hyderabad. The design focuses on elegant dessert branding, a cozy café atmosphere, smooth user experience, and visually appealing content that attracts customers and enhances brand identity.
